# 令和７年度 千葉県児童生徒・教職員科学作品展実施要項

## 審査のポイント（科学工夫作品の部）
以下の観点で審査されます：
- 着想が新しいか
- 創意工夫が盛り込まれているか
- 研究努力が積まれているか
- 学習したことを発展させているか

## 🏆 賞の種類（一部抜粋）
- 千葉県知事賞（小・中・高 各1点）
- 千葉県教育長賞
- 千葉市教育長賞
- 発明協会会長奨励賞
- 千葉県総合教育センター所長賞　など
優秀作品は全国展への推薦対象にもなるようです！

## 📅 スケジュール概要
- 作品受付：9月25日（木）
- 審査：9月29日〜10月1日
- 一般公開：10月18日（土）〜19日（日）
- 作品返却：10月20日（月）
